# Engineering Support Automation Toolkit

## Project Overview

The Engineering Support Automation Toolkit is a simple portfolio project that simulates the day-to-day tasks of an Engineering Support Virtual Assistant.

The project focuses on:

- Project tracking
- Document control
- Excel automation
- VBA macros
- PowerShell automation
- Blockpad calculations
- Workflow automation

## Objectives

- Practice Microsoft Excel advanced features
- Learn VBA and Macros
- Explore Blockpad calculations
- Build PowerShell automation scripts
- Create a simple engineering support workflow
