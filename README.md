## Project Workflow

The project demonstrates a simple engineering reporting workflow:

1. Project information is maintained in Excel.
2. The dashboard automatically summarizes project status and KPIs.
3. A VBA macro refreshes the dashboard and exports a PDF report.
4. Power Automate Desktop launches Excel and executes the reporting workflow.
5. A PowerShell script archives the generated report and creates an execution log.

This project was created to demonstrate my approach to learning new technologies through practical application.
