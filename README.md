# Engineering Support Automation Toolkit

A portfolio project designed to simulate the day-to-day responsibilities of an Engineering Support Virtual Assistant.

## Project Goals

This project is being developed to strengthen practical skills in:

- Microsoft Excel
- VBA & Macros
- Pivot Tables
- Structured Workbooks
- Data Validation
- Blockpad
- PowerShell
- Microsoft Power Automate
- Technical Documentation
